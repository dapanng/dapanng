## 🎯 玩法介绍

这个是我为了和朋友玩而开发的mod，这里面一共开发了9个游戏玩法（具体玩法内容可以看下文内容），由于是从零开始自学做mod，并且主要是自用，所以可能会存在一些小bug，我已经将游玩过程中发现的恶行bug都尽可能修复，以下就是里面包含的玩法内容

### 1. 献祭玩法 

> 所有玩家命运相连，以身体为代价博取一线生机。

- 所有玩家**共享生命值和饥饿度**
- 跳跃次数达到阈值时，为所有玩家身边**生成敌对生物**
- 玩家可**献祭身体部位**（左臂/右臂/左腿/右腿/心脏）进行抽奖，抽奖结果影响所有玩家的最大生命值
- 每次死亡会**永久减少最大生命值**
- 献祭部位后获得对应 debuff：
  - 左臂 → 挖掘疲劳
  - 右臂 → 虚弱
  - 左腿 → 缓慢
  - 右腿 → 缓慢
  - 心脏 → 凋零
- 消耗**钻石**可赎回身体部位
- 跳跃阈值和赎回费用可在 UI 调节

---

### 2. 纸人玩法 

> 如纸般脆弱，遇水即溶，靠近火焰则灰飞烟灭。

- 玩家初始最大生命值为 **1**
- 可食用**纸、雪块、冰块、蓝冰、浮冰**恢复状态
- 接触**水、岩浆、火焰**立即死亡
- 只能穿戴**皮革装备**
- 背包中的**岩浆桶会被丢弃**
- 受到伤害按**倍率放大**
- 永久获得**缓降效果**
- **耐力值机制**：靠近热源消耗耐力，冻结时不消耗，耗尽则燃烧
- 用镐子挖掘冰块有概率直接掉落**整冰块**

---

### 3. 残疾玩法 

> 每一次受伤都可能让你永远失去一个背包格子。

- 玩家每次受伤达到阈值时，按概率**禁用一个背包格子**
- 被禁用的格子放入**屏障方块**作为视觉标识
- 无法在禁用格子中放置物品
- 受伤阈值和禁用概率可在 UI 调节

---

### 4. 肉鸽玩法 

- 玩家每进入新区块时，**随机生成指定数量的生物**
- 只有击杀所有生成的生物后才能**离开该区块**
- 生物数量可在 UI 调节

---

### 5. 加速玩法 

- 除玩家外的所有生物**行动速度加快**
- 加速倍率可在 UI 调节

---

### 6. 全随机玩法 

> 一切皆有可能，一切都在变化。

- 所有**合成配方被打乱**
- 击败生物和挖掘方块的**掉落物被打乱**
- 每隔一定时间**自动重新随机**
- 随机化间隔可在 UI 调节

---

### 7. 追击玩法 

> 万物皆敌，无处可逃。

- **500格内**所有生物会追杀玩家
- 和平生物也会获得**攻击伤害**并追击玩家
- 停止玩法后生物恢复正常
- 和平生物攻击范围可在 UI 调节（默认 1.0，范围 0.5-32.0）

---

### 8. 传送玩法 

> 时空错乱，你永远不知道下一秒会在哪里。

- 玩家每隔一定时间被**随机传送**到任意维度
- 维度概率：主世界 60%、地狱 30%、末地 10%
- 坐标范围：主世界 ±10000，地狱 ±300
- 自动寻找安全 Y 坐标（地狱跳过基岩层）

---

### 9. 弓箭手大作战

> 远程为王，箭雨如梭，在骷髅的围攻中求生。

- 玩家只能使用**远程武器**（弓、弩、剑），不可穿戴盔甲
- 世界中所有非目标生物被替换为远程敌对生物（骷髅/流浪者/女巫/溺尸/守卫者/烈焰人/恶魂）
- 骷髅/流浪者装备**钻石头盔 + 力量5弓 + 力量5效果**
- 溺尸装备**引雷3 + 穿刺5三叉戟 + 保护4海龟壳 + 抗火效果**
- 女巫永久**抗性提升3**
- 骷髅射出的箭有概率带 **TNT 跟随**（概率可调）
- 流浪者射出的箭有概率带**凋零效果**（概率可调）
- 箭在水中**不受衰减**
- 击杀怪物积累**护甲值**（死亡不清空）
- 末影龙增强（末影水晶重生）
- 合成配方修改（移除剑/斧/盔甲配方，末影之眼需恶魂之泪）
- 生物生成上限可在 UI 调节（默认 120，范围 10-500）
- 武器附魔等级可在 UI 调节

## 🕹️ 使用方法

1. 在游戏中按 **V** 键打开玩法选择界面
2. 选择想要游玩的玩法模式
3. 点击**开始**按钮启动玩法，点击**停止**按钮关闭玩法
4. 每种玩法有独立的**参数配置界面**，可根据需要调整
